About PneumoAI

Deep Learning for Medical Image Analysis

Project Overview

PneumoAI is an end-to-end deep learning project that leverages state-of-the-art convolutional neural networks to detect pneumonia from chest X-ray images. This project demonstrates the practical application of artificial intelligence in medical diagnostics.

Pneumonia is a serious lung infection that affects millions of people worldwide. Early and accurate detection is crucial for effective treatment. This project aims to assist medical professionals by providing rapid, AI-powered preliminary screening of chest X-rays.

Dataset Information

Chest X-Ray Images (Pneumonia)

Source: Kaggle - Paul Mooney

Total Images: 5,856 chest X-ray images

Classes:

  • NORMAL: 1,583 images (27%)
  • PNEUMONIA: 4,273 images (73%)

Image Format: JPEG

Split: Train (5,216) / Validation (16) / Test (624)

Data Preprocessing
  • Image Resizing: All images resized to 224×224 pixels
  • Normalization: Pixel values scaled to [0, 1] range
  • Data Augmentation: Rotation, shifting, zoom, and horizontal flip
  • Class Weighting: Balanced to handle class imbalance

Model Architecture

Transfer Learning Approach

This project utilizes VGG16, a powerful convolutional neural network pre-trained on ImageNet (1.4 million images, 1000 classes). Transfer learning allows us to leverage the feature extraction capabilities learned from millions of images and fine-tune them for our specific pneumonia detection task.

Network Layers
  1. VGG16 Base Model (frozen weights)
    • 13 Convolutional layers
    • 5 Max Pooling layers
    • Pre-trained on ImageNet
  2. Global Average Pooling - Reduces spatial dimensions
  3. Dense Layer (128 neurons, ReLU) - Feature combination
  4. Dropout (0.5) - Prevents overfitting
  5. Output Layer (1 neuron, Sigmoid) - Binary classification
Training Configuration
  • Optimizer: Adam (Adaptive learning rate)
  • Loss Function: Binary Crossentropy
  • Batch Size: 32 images per batch
  • Epochs: 10 (with early stopping)
  • Early Stopping: Patience of 5 epochs on validation loss

How It Works

Prediction Pipeline
  1. Image Upload: User uploads a chest X-ray image through the web interface
  2. Preprocessing: Image is resized to 224×224 and normalized
  3. Feature Extraction: VGG16 base model extracts high-level features
  4. Classification: Custom layers process features and predict probability
  5. Result: Output shows prediction (Normal/Pneumonia) with confidence score
Understanding the Output

The model outputs a probability score between 0 and 1:

  • Probability > 0.5: Classified as PNEUMONIA
  • Probability < 0.5: Classified as NORMAL

The confidence score represents how certain the model is about its prediction. Higher confidence indicates stronger evidence for the classification.
